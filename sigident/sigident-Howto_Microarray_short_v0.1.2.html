sigident - Howto Microarray (short)

An alternative workflow: four generic functions for signature identification

In order to simplify the whole sigident-workflow described in the vignette sigident - Howto Microarray, we wrapped all functions of the sigident R package framework into four big functions.

Preprocessing

In order to use this R package and its functions, you need to prepare a merged gene expression dataset. The workflow to achieve this is presented in the following by making use of the R package sigident.preproc.

For a more detailed background and description of the following steps please view the sigident.preproc packageâ€™s vignette.
